Module axi_cdc_dst

Destination-clock-domain half of the AXI CDC crossing.

For each of the five AXI channels, this module instantiates the source or destination half of a CDC FIFO. IMPORTANT: For each AXI channel, you MUST properly constrain three paths through the FIFO; see the header of cdc_fifo_gray for instructions.

Parameters

LogDepth: int unsigned

Depth of the FIFO crossing the clock domain, given as 2**LOG_DEPTH.
