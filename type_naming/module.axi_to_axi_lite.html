Module axi_to_axi_lite

An AXI4+ATOP to AXI4-Lite converter with atomic transaction and burst support.

Parameters

AddrWidth: int unsigned

DataWidth: int unsigned

IdWidth: int unsigned

UserWidth: int unsigned

MaxWriteTxns: int unsigned

MaxReadTxns: int unsigned

FallThrough: bit

axi_req_t: type

axi_rsp_t: type

axi_lite_req_t: type

axi_lite_rsp_t: type

Ports

clk_i: input logic

rst_ni: input logic

test_i: input logic

sbr_port_req_i: input axi_req_t

sbr_port_rsp_o: output axi_rsp_t

mgr_port_req_o: output axi_lite_req_t

mgr_port_rsp_i: input axi_lite_rsp_t

Signals

filtered_req: axi_req_t

splitted_req: axi_req_t

filtered_rsp: axi_rsp_t

splitted_rsp: axi_rsp_t
