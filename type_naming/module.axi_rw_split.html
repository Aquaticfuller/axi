Module axi_rw_split

Splits a single read / write subordinate into one read and one write manager

Connects the ar and r channel of the read / write subordinate to the read manager and the aw, w and b channel of the read / write subordinate to the write manager

Parameters

axi_req_t: type

axi_rsp_t: type

Ports

clk_i: input logic

rst_ni: input logic

sbr_req_i: input axi_req_t

sbr_rsp_o: output axi_rsp_t

mgr_read_req_o: output axi_req_t

mgr_read_rsp_i: input axi_rsp_t

mgr_write_req_o: output axi_req_t

mgr_write_rsp_i: input axi_rsp_t
