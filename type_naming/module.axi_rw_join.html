Module axi_rw_join

Joins a read and a write subordinate into one single read / write manager

Connects the ar and r channel of the read subordinate to the read / write manager and the aw, w and b channel of the write subordinate to the read / write manager

Parameters

axi_req_t: type

axi_rsp_t: type

Ports

clk_i: input logic

rst_ni: input logic

sbr_read_req_i: input axi_req_t

sbr_read_rsp_o: output axi_rsp_t

sbr_write_req_i: input axi_req_t

sbr_write_rsp_o: output axi_rsp_t

mgr_req_o: output axi_req_t

mgr_rsp_i: input axi_rsp_t
